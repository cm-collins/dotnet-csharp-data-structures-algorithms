## Problem in Focus
**14. Longest Common Prefix (Easy)**  
Given an array of strings, return the longest common prefix. If none, return `""`.

**Examples**
- `["flower","flow","flight"]` → `"fl"`
- `["dog","racecar","car"]` → `""`

**Constraints**
- `1 <= strs.length <= 200`
- `0 <= strs[i].length <= 200`
- lowercase English letters only (if non-empty)
